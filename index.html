What is a <div>?

A div tag has no specific meaning, it is simply a way to group tags. In order to give a div some style, you'll need to add properties such as:

  • height
  • width
  • margin
  • padding
  • border
  • background-color

Use different border styles and colors when designing your page to keep track of all the different divs. You can always delete the border later.

The green border you see is a div with the class name "container" and it's used for the <h1>. There is also a the same class of div with the <h2>. The style is: 

        .container {
          border: 3px solid green;
          margin: 0 auto;
          width: 50%;
        }
Notice how the width is 50%, so it will only take up 50% of the width of whatever it contained in.

The red border you see is a div with an id (#main) and it is styled with: 

        #main {
          margin: auto;
          border: 3px solid red;
          width: 90%;
          padding: 5%;
        }

The blue border you see is shows the body tag. The styling for the body is: 

        body {
          background-color: silver;
          font-family: 'Inconsolata', monospace;
          text-align: center;
          border: 5px solid blue;
          margin: 0;
        }

Margin vs. Padding

  • Margin is to separate blocks from things outside
  • Padding is to separate blocks from things inside

The blue border has no margin, so that's why it's going to the very edge of the page.

The red border has padding so there is some space between the text and the border.

Units of Measurement

Unit Description
px (absolute) pixels (1px = 1/96th of 1in)
em (relative) Relative to the font-size of the element (2em means 2 times the size of the current font)
rem (relative) Relative to font-size of the root element
vh (relative) Relative to 1% of the height of the viewport
vw (relative) Relative to 1% of the width of the viewport
